# SxDevOps 用户管理

## 一、概述

用户管理模块提供平台用户的创建、编辑、角色分配和状态管理。

**入口：** 侧边栏 → 系统管理 → 用户管理 → 用户列表 Tab
**权限：** `rbac.user.view` / `rbac.user.manage`

## 二、用户列表

| 列 | 说明 |
|----|------|
| 用户名 | 登录账号 |
| 邮箱 | 关联邮箱（用于通知） |
| 角色 | 分配的角色标签 |
| 状态 | 启用 / 停用 |
| 创建时间 | 账号创建时间 |

## 三、创建用户

点击"新增用户"：

| 字段 | 说明 |
|------|------|
| 用户名 | 登录账号（必填，唯一） |
| 邮箱 | 用于通知和找回密码 |
| 密码 | 初始密码 |
| 角色 | 选择一个或多个角色 |
| 激活状态 | 启用 / 停用 |

## 四、编辑用户

- 修改邮箱、角色
- 重置密码
- 启用/停用账号
- 停用后用户无法登录，现有 Token 立即失效

## 五、默认演示账号

执行初始化数据后，系统预设以下账号（密码 `Admin@123456`）：

| 账号 | 角色 | 说明 |
|------|------|------|
| admin | 平台管理员 | 全部权限 |
| ops_demo | 运维工程师 | 运维操作 |
| dev_demo | 开发工程师 | 可观测性查看 |
| audit_demo | 审计员 | 只读审计 |
| viewer_demo | 观察者 | 最小只读 |

> 生产环境请立即修改或禁用演示账号。
