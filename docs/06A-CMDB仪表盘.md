# SxDevOps CMDB 仪表盘

## 一、概述

CMDB 仪表盘展示配置管理数据库的整体概览——CI 类型分布、配置项统计、资源申请状态等。

**入口：** 侧边栏 → CMDB → CMDB 总览
**权限：** `cmdb.dashboard.view`

## 二、统计卡片

| 卡片 | 说明 |
|------|------|
| CI 类型数 | 已定义的 CI 类型数量 |
| 配置项总数 | 所有 CI 实例总数 |
| 关联关系数 | CI 之间的关联关系数量 |
| 待审批申请 | 资源申请中待审批的数量 |

## 三、数据来源

- CI 类型和配置项：`CIType` 和 `ConfigItem` 表
- iTop 同步的外部 CI 也统一计入
- Zabbix 设备映射关联的配置项会标记监控状态
