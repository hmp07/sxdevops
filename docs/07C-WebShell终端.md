# SxDevOps WebShell 终端

## 一、概述

WebShell 提供基于浏览器的 SSH 终端，可直接在平台上操作远程主机，无需额外工具。

**入口：** 通过任务工作台或资源底座点击主机 → WebShell，或直接访问 `/webshell/:hostId`
**权限：** `ops.host.terminal`

## 二、使用方式

### 2.1 打开终端

1. 进入 资源底座 → 选择主机
2. 点击"WebShell"按钮
3. 新标签页打开终端界面

### 2.2 终端功能

- 基于 xterm.js 的全功能终端模拟器
- 支持复制/粘贴
- 自适应窗口大小
- ANSI 颜色和光标控制
- 会话保持

## 三、前置条件

| 条件 | 说明 |
|------|------|
| 主机 SSH 配置 | 主机需配置 SSH 端口、用户名、密码/密钥（在资源底座中维护） |
| 网络可达 | 平台后端需能 SSH 连接到目标主机 |
| 权限 | 需要 `ops.host.terminal` 权限 |

## 四、安全说明

- SSH 连接通过平台后端代理（Daphne WebSocket → Paramiko SSH）
- 密码在数据库中以明文存储（建议使用密钥认证）
- 所有终端操作记录在平台的 SSH 消费者日志中
- 支持多会话并行
