# SxDevOps 平台总览与仪表盘

## 一、概述

运行概览仪表盘是 SxDevOps 的首页，展示平台整体运行态势——告警统计、AIOps 使用情况、模型调用成本等关键指标。

**入口：** 侧边栏 → **运行概览**（默认首页）
**权限：** `ops.dashboard.view`

## 二、页面布局

### 2.1 时间范围选择器

顶部提供时间范围筛选，支持自定义开始/结束时间。"全部时间"按钮重置筛选。

### 2.2 统计卡片

| 卡片 | 说明 |
|------|------|
| MCP 工具调用 | AI 智能体调用 MCP 工具的次数 |
| Skill 命中 | AI 智能体匹配到技能包的次数 |
| Action 命中 | AI 智能体识别到的动作类型次数 |
| 模型成本 | 模型调用次数、Token 用量、费用、平均耗时 |

### 2.3 主机与告警统计

- 主机总数、在线数、离线数
- 告警总数、活跃告警、严重告警
- 部署统计（总数、运行中、失败）
- Zabbix 数据源数量和告警数量

## 三、数据刷新

点击"刷新"按钮手动更新。仪表盘数据来源于：
- 主机/部署/告警：数据库实时查询
- AIOps 统计：智能体审计记录聚合
- Zabbix：`ZabbixDataSource` 表 + Alert 表 `source_type='zabbix'` 统计

## 四、与告警中心联动

仪表盘的告警统计卡片点击后跳转到告警中心对应筛选视图（活跃/严重/已屏蔽）。
