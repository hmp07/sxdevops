# SxDevOps 资源拓扑

## 一、概述

资源拓扑以可视化图形展示 CMDB 中 CI 之间的关联关系，按业务线和环境分层布局。

**入口：** 侧边栏 → CMDB → 资源拓扑
**权限：** `cmdb.topology.view`

## 二、拓扑视图

### 2.1 布局逻辑

```
业务线（行）
  └── 环境（列）
        └── CI 类型
              └── 配置项节点
```

### 2.2 节点

每个节点代表一个配置项，颜色按 CI 类型区分。节点显示名称和类型图标。

### 2.3 连线

连线代表 CI 之间的关联关系，不同类型用不同颜色和样式：
- 运行关系（runs_on）
- 依赖关系（depends_on）
- 连接关系（connects_to）

## 三、交互

- 拖拽节点调整位置
- 点击节点查看详情
- 滚轮缩放画布
- 按业务线、环境筛选显示范围

## 四、与 iTop 的关系

如果配置了 iTop 数据源并执行了同步，iTop 中的 CI 和关联关系会出现在拓扑图中，与手动录入的 CI 统一展示。
